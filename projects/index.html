Improve stability and feature set of Kexi MS Access import

Ongoing

Mentor: Jaroslaw Staniek
Kexi is a visual database applications creator, part of the Calligra Suite. Kexi Support for importing Microsoft Access databases (2003 or older) is built into Kexi. It uses the import wizard by selecting desired .mdb file. MDB databases are files produced by the Microsoft 'Jet' database engine. They are used as the format for Microsoft Access databases. MDB Tools is an open source suite of libraries and utilities to read (and soon write) MDB database files. Current mdbtools sources included with Kexi (kexi.git) for MS Access mdb format support can be outdated. Trying to get people to move to Kexi from Access at some point, means the should be able to move their existing data/design also.

Undergraduate Thesis: Stable Matching problem under Social stability

Ongoing

Guide: Suhel Ahamed, IT Department, GGU, Bilaspur
The question we are interested in analyzing is the following: Given a set of preference lists for all men and women, one per- son is in another persons realm of possible spouses if there is a stable matching in which the two people are married. A persons optimal spouse is their most pre- ferred person within their realm of possibility. A persons pessimal spouse is their least preferred person in their realm of possibility. TMA marries every man to his optimal spouse. TMA marries every woman to her pessimal spouse.
